# 배포 정보

- GitHub 저장소: contentsmining/shorts-landing
- Vercel 프로젝트: contentsminings-projects 팀 소속
- 배포 주소(프로덕션): https://shorts-landing-psi.vercel.app

## 참고
- `main` 브랜치에 push하면 Vercel이 자동으로 위 주소에 재배포한다.
- 로컬 `vercel` CLI는 `glownova` 계정으로 로그인되어 있어 이 프로젝트가 CLI에서는 보이지 않는다.
  Vercel 대시보드에서 확인하려면 `contentsmining` 계정/팀으로 로그인해야 한다.
- git push 권한도 `contentsmining` GitHub 계정 기준이다 (`gh auth switch --user contentsmining` 필요할 수 있음).
